7-Day Roadmap

Day Focus Topics Use Cases
Day 1 AWS Foundations Regions, AZs, IAM, roles, CLI, S3 Secure access, storage, and AWS basics.
Day 2 Networking & Compute VPC, subnets, IGW, NAT, route tables, SG, NACL, EC2 Design and secure cloud infrastructure.
Day 3 CI/CD Foundations CodeCommit, Artifact, pipeline concepts Understand release workflows.
Day 4 CodeBuild buildspec.yml, environments, artifacts, logs Automate builds and tests.
Day 5 CodeDeploy AppSpec, deployment groups, hooks, blue/green Safely deploy to servers or containers.
Day 6 CodePipeline Source → Build → Deploy, approvals, SNS Orchestrate full automation.
Day 7 Real Project CodeCommit → CodeBuild → CodeDeploy → EC2 Build a working CI/CD pipeline.

Core Services

  • IAM: Users, groups, policies, and roles. IAM is the main AWS access control service. [web:1][web:10]
  • S3: Object storage for artifacts, logs, backups, and static websites.
  • EC2: Virtual servers for application workloads.
  • VPC: Private network design with subnets and routing.
  • CodeBuild: Managed builds and tests. AWS describes it as a fully managed build service. [web:6]
  • CodePipeline: Automates build, test, and deploy phases after code changes. [web:5][web:2]

Project Ideas

  • EC2 CI/CD pipeline.
  • Container pipeline with ECR and ECS.
  • Serverless pipeline with Lambda and SAM.
  • Multi-environment dev/staging/prod pipeline.
